Selectors - Radio buttons and checkboxes: Alpha design pattern

Default browser radio buttons and checkboxes are small and difficult to see for low vision users. Selections should be visually clear and easier to distinguish.

When to use

Larger radio buttons and checkboxes should be used everything. They will automatically be updated when using fieldflow. When not using fieldflow, you will need to use the code found below for the customized design.

Implementation

Radio buttons

Use the following HTML to create radio buttons:

<div class="cst-inputs radio-container">
	    <label>Label 1
	        <input type="radio">
	        <span></span>
	    </label>
	    <label>Label 2
	        <input type="radio">
	        <span></span>
	    </label>
			etc..
</div>

Checkboxes

Use the following HTML to create checkboxes:

<div class="cst-inputs checkbox-container">
	    <label>Label 1
	        <input type="checkbox">
	        <span></span>
	    </label>
	    <label>Label 2
	        <input type="checkbox">
	        <span></span>
	    </label>
			etc..
</div>
