H1 subtitle: Alpha design pattern

Some pages have a subtitle that describes a category the page belongs to. Render that subtitle in a way that more clearly shows it's part of that category, making the title easier to read.

When to use

Many pages fit into a section or category of other pages. To make the context of the page clear, we often make that section or category part of the title. For example:

How to apply - EI maternity and parental benefits

This is important because even though we want to focus on "How to apply", we need it to be clear that maternity and parental benefits are what are being applied to.

Use when you would write the main page heading with a subtitle that describes the category or section the page belongs to.

What to avoid

Not sure what might be an issue here.

Implementation

A single h1 is used with spans to separate the 3 components:

  • the page name, which should be read 1st
  • the separator, which should be a hyphen
  • the subtitle, which should be read last

Option 1

The following is putting the class .stacked on the root h1, which then requires the stubby line to be replaced on the first child

<h1 property="name" id="wb-cont" dir="ltr" class="stacked">
	<span>H1 subtitle</span> - <span>Alpha design pattern</span>
	</h1>

The CSS required is as follows

h1#wb-cont.stacked {
display: flex;
flex-direction: column-reverse;
visibility: hidden;
}
h1.stacked span:nth-child(1), h1.stacked span:nth-child(2) {
visibility: visible;
}
h1.stacked span:nth-child(2) {
font-size: 0.8em;
color: #666;
font-weight: 200;
padding-bottom: 0.175em;
}
h1.stacked span:nth-child(1) {
margin-top: -1.1em;
padding-bottom: 0.2em;
border-bottom: .18em solid #af3c43;
-o-border-image: linear-gradient(to right, #af3c43 71px, transparent 71px);
border-image: linear-gradient(to right, #af3c43 71px, transparent 71px);
border-image-slice: 1;
}

Option 2

The following is creating a parent class for the class span.stacked , which does not require the the stubby line to be re-added

<h1 property="name" id="wb-cont" dir="ltr">
  <span class="stacked"
    <span>Page name</span> - <span>Subtitle</span>
  </span>
</h1>

The CSS required is as follows

h1 span.stacked {
display: flex;
flex-direction: column-reverse;
visibility: hidden;
}
h1 span.stacked span:nth-child(1), h1 span.stacked span:nth-child(2) {
visibility: visible;
}
h1 span.stacked span:nth-child(2) {
font-size: 0.8em;
color: #666;
font-weight: 200;
margin-bottom: 0.175em;
}
h1 span.stacked span:nth-child(1) {
margin-top: -1.1em;
}
